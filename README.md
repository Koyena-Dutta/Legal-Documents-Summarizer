## 📌 Installation
Clone the repository and install dependencies.
```bash
# Clone repo
git clone https://github.com/Koyena-Dutta/Legal-Documents-Summarizer.git

# Go into project folder
cd Legal-Documents-Summarizer/frontend

# Install dependencies
npm install

# Start dev server
npm run dev
```
